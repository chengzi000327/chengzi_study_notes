## 一、认知进化：从"外挂提示词"到"神经网络硬编码"

传统 ReAct 架构依赖外部长 Prompt 约束，在 7B 中型模型上存在致命缺陷：长上下文迷失、格式崩溃、幻觉调用。

PersonaVLM 的核心突破：利用 **后训练强化学习（Post-training RL）**，将整个 ReAct 工具检索流直接固化/焊死为模型的参数条件反射。

### 神经层面的格式与逻辑规训（三个铁律奖励函数）

- **格式惩罚（Format Gate）：** 严格封锁 `<think>`、`<retrieve>`、`<answer>` 闭合标签。不按标准输出在梯度反向传播时直接被一票否决（零奖励）。
- **逻辑锚定（Consistency Constraint）：** 思考链（CoT）内容必须在数学上成为检索动作的"语义前置条件"。想查什么，必须先推理为什么查。
- **事实倒逼（Accuracy Incentive）：** 构造非检索不可得的硬核常识事实 QA，逼迫模型在参数空间内建立"模糊意图 → 本能触发检索"的依赖路径。

## 二、存储哲学：心理学架构与计算机工程的数据结构映射

长周期动态个性化的最大挑战：记忆臃肿（Context Explosion）与冲突（Information Contraction）。

论文将认知心理学对人类记忆的分类映射到计算机科学的数据结构策略（Mutable vs. Immutable）：

### 四类记忆的数据结构设计

**1. 核心记忆（Core Memory）**
- 心理定位：自我与客体认知
- 数据结构：结构化固定 Key-Value 字典
- 读写策略：覆盖写（Mutable），消灭冲突
- 加载机制：全局常驻变量（无需检索）
- 触发条件：用户修改或透露了最根本的社会/人设属性

**2. 程序记忆（Procedural Memory）**
- 心理定位：潜意识反射与预期
- 数据结构：状态机标识（Identifier）+ 第三人称短句
- 读写策略：智能归纳合并与更新（Mutable）
- 核心作用：行为预测与周期的主动配合

**3. 语义记忆（Semantic Memory）**
- 心理定位：百科知识谱系
- 数据结构：密集向量 + 明文 + 时间戳 + 视觉切块（Image Crop）
- 读写策略：纯追加（Additive）+ 时间衰减检索加权
- 核心作用：跨时间的偏好锚定（如换口味的软逻辑解决）

**4. 情景记忆（Episodic Memory）**
- 心理定位：时光编年史
- 数据结构：双层解耦（索引摘要层 + 原始日志层）
- 读写策略：历史封存，绝对只读不可篡改（Immutable）
- 核心作用：人际关系动力学变迁分析

### 冲突消灭与消化的双轨制

1. **硬物理删除（源头断绝）：** Core 和 Procedural 发生变动直接修改最新 Canonical Version，从工程上消灭冲突。
2. **软逻辑消化（大脑解耦）：** Semantic 和 Episodic 只增不删。
   - 时间衰减加权：检索时 $[\text{Similarity} \times \text{Time Decay Factor}]$，新事实排位靠前。
   - LLM 软化矛盾：模型同时查出旧偏好和新偏好时，由 RL 训练出的 `<think>` 大脑进行时序逻辑判断，主动吐故纳新。

## 三、算法演进：算力内卷下的 RL 核心策略博弈

### PPO（绝对值导师制 · 旗舰标配）
- 现状：GPT-4o, Claude 3.7, Gemini Pro 等闭源巨头的底牌
- 本质：算力管够时，绝对价值估算在开放式生成中最稳健
- 痛点：需要 Critic 模型绝对打分，显存极重（常驻 4 个大模型）

### GRPO（组内相对排名 · 推理利器）
- 现状：DeepSeek-R1 核心基石，PersonaVLM 用来实现 7B 本地模型标签约束
- 本质：拿掉 Critic 模型，显存内零成本"组内相对论"
- 缺点：全组摆烂（全错）时失效

### GSPO（尖子生淘汰赛 · 逻辑硬核）
- 现状：大厂攻克顶级数学、高阶代码、黑盒环境规划时的前沿变体
- 本质：解决长思维链探索中"大量错误样本拉低全组平均分"的痛点
- 核心：稀疏过滤器将平庸回答权重归零，强制模型只向 Top-k 学习

### DPO（静态偏好映射 · 开源主流）
- 现状：Llama 3 系列微调、低成本平替
- 本质：将 RL 损失等价压缩到离线 SFT 阶段
- 代价：模型永远失去"自主探索（Exploration）"和"自我觉醒思维链"的上限

## 结语

用轻量级硬件实现长效、高情商、懂纪律的个人专属伴侣大模型的完整闭环方案。

PersonaVLM 证明了：大模型的未来不单单是参数量的暴力堆叠，通过格式化的 GRPO 算法重塑大脑，配合心理学级的四轨数据结构约束，一个 7B 的小模型同样能开辟出让通用巨头感到战栗的"高同理心"本地化智能新世界。
